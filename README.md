# AI 邮件智能分析

一个功能强大的邮件智能分析工具，支持批量分析、高价值识别、AI自动回复等功能。

## 功能特性

### 核心功能
- **邮件批量分析**：支持多账号批量分析邮件
- **高价值识别**：智能识别有价值的邮件
- **退订邮件识别**：自动识别退订请求
- **AI自动回复**：基于大模型自动生成回复内容

### 本地邮件分析
- 支持 `.eml` 格式邮件解析
- 支持 `.mbox` 格式邮件解析
- 支持 `.msg` 格式邮件解析

### 其他功能
- **WhatsApp接入**：集成WhatsApp机器人
- **统计分析**：详细的数据统计和分析
- **下载管理**：邮件附件下载管理
- **知识库管理**：知识库功能
- **模型设置**：支持多模型配置
- **账户管理**：多账户管理
- **卡密验证**：设备绑定的卡密验证系统
- **主题切换**：支持浅色/深色主题
- **定时任务**：定时自动分析


## 通知和更新

- 点击仪表盘右上角的通知图标查看最新通知
- 支持版本检查和更新提示
- 通知数据从GitHub获取

## 技术栈

- **GUI框架**：PyQt5
- **邮件处理**：imap-tools
- **数据库**：SQLite
- **数据分析**：pandas, openpyxl
- **HTTP请求**：requests
